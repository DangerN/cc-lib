# cc-lib

[![GitHub release](https://img.shields.io/github/release/DangerN/cc-lib.svg)](https://github.com/DangerN/cc-lib/releases)

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     cc-lib:
       github: DangerN/cc-lib
   ```

2. Run `shards install`

## Usage

```crystal
require "cc-lib"
```
